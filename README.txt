
A SWI-Prolognak telepítve kell lennie a gépen.

Az alábbi környezeti változókat kell beállítani (felhasználói változó):

NEVE:	ÉRTÉKE:					Megjegyz.:
------------------------------------------------------------------
PATH	C:\Program Files (x86)\swipl\bin	A prolog telepítési könyvtárának bin alkönyvtára.
SWI_HOME_DIR	C:\Program Files (x86)\swipl	A prolog telepítési könyvtára.

///////////////////////////////////////////////////////////////////////////////////////////////////////////

SWI-Prolog must be installed on the computer.

You have to set these envirotment variables (User variable).


NAME:	VALUE:					Comment:
------------------------------------------------------------------
PATH	C:\Program Files (x86)\swipl\bin	SWI-Prolog install directorys /bin subdirectory. 
SWI_HOME_DIR	C:\Program Files (x86)\swipl	SWI_Prolog install directory.
